# Icons Folder

This folder contains the app icons for the PWA.

## Required Files
- `icon-192.png` - 192x192 pixel icon
- `icon-512.png` - 512x512 pixel icon
- `favicon.ico` - 32x32 pixel favicon

## How to Generate Icons

### Option 1: Use the Icon Generator (Easy)
1. Open `generate-icons.html` in your browser
2. Click each download button to generate the icons
3. Save the files in this folder

### Option 2: Use an Online Tool
1. Go to https://realfavicongenerator.net/ or similar tool
2. Upload the `icon.svg` file
3. Download the generated icons
4. Place them in this folder

### Option 3: Create Custom Icons
- Use any image editor (Photoshop, GIMP, Figma, Canva, etc.)
- Create colorful, simple designs suitable for kids
- Export as PNG files at the required sizes
- Name them correctly: `icon-192.png`, `icon-512.png`, `favicon.ico`

## Design Guidelines
- Keep it simple and recognizable at small sizes
- Use bright, cheerful colors
